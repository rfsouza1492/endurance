# Endurance Engine Room

Backend API for Endurance Engine Room.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env` and configure:
```bash
cp .env.example .env
```

3. Build:
```bash
npm run build
```

4. Run:
```bash
npm start
```

For development:
```bash
npm run dev
```
